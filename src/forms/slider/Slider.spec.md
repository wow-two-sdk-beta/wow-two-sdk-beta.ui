# Slider

## Purpose
Single-value range slider. Native `<input type="range">` cross-browser-styled. Range / dual-thumb variant lives at L5.

## Props
| Name | Type | Default |
|---|---|---|
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` |
| `min` | `number` | `0` |
| `max` | `number` | `100` |

## Dependencies
Foundation: `utils/cn`, `primitives/formControlContext`.
