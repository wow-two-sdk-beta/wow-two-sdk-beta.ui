# PasswordInput

## Purpose
`<input type="password">` with optional visibility toggle. Default `autoComplete="current-password"` — pass `"new-password"` for sign-up forms.

## Props
| Name | Type | Default |
|---|---|---|
| `toggleable` | `boolean` | `true` |

## Dependencies
Foundation: `utils/cn`, `icons/Icon`, `primitives/formControlContext`. Same-domain: `forms/InputStyles`.
