# Radio

## Purpose
Single radio button. Use multiple with the same `name` to form a group; for arrow-key navigation use `RadioGroup` (L4).

## Props
| Name | Type | Default |
|---|---|---|
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` |

## Dependencies
Foundation: `utils/cn`, `primitives/formControlContext`.
