# SwitchField

## Purpose
Switch + label + optional description in one clickable `<label>`. Common settings-row UI.

## Props
| Name | Type | Default |
|---|---|---|
| `label` | `ReactNode` | — (required) |
| `description` | `ReactNode` | — |
| `side` | `'left' \| 'right'` | `'left'` |
| ...rest | `SwitchProps` | — |

## Dependencies
Foundation: `utils/cn`, `hooks/useId`. Same-domain: `Switch`.
