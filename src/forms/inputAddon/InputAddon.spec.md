# InputAddon

## Purpose
Wrap any input with leading and/or trailing addon slots — connected visually to the input. Common for "https://", ".com", units ("kg", "px").

## Props
| Name | Type | Default |
|---|---|---|
| `leading` | `ReactNode` | — |
| `trailing` | `ReactNode` | — |
| `children` | `ReactNode` (single input) | — (required) |

## Dependencies
Foundation: `utils/cn`.
