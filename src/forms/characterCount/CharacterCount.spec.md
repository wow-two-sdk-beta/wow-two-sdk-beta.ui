# CharacterCount

## Purpose
Counter beneath a length-limited input. Renders `current / max` (or just `current`). Switches to destructive color when over.

## Props
| Name | Type | Default |
|---|---|---|
| `value` | `number` | — (required) |
| `max` | `number` | — (required) |
| `showMax` | `boolean` | `true` |

## Dependencies
Foundation: `utils/cn`.
