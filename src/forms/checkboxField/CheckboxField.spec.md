# CheckboxField

## Purpose
Checkbox + right-side label + optional description, wrapped in a `<label>` so clicking text toggles.

## Props
| Name | Type | Default |
|---|---|---|
| `label` | `ReactNode` | — (required) |
| `description` | `ReactNode` | — |
| ...rest | `CheckboxProps` | — |

## Dependencies
Foundation: `utils/cn`, `hooks/useId`. Same-domain: `Checkbox`.
