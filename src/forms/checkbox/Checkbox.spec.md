# Checkbox

## Purpose
Native `<input type="checkbox">` with custom visual. Strict atom — no built-in label. Pair with `Label` via `FormControl`, or wrap manually.

## Props
| Name | Type | Default |
|---|---|---|
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` |
| `indeterminate` | `boolean` | `false` |

## Dependencies
Foundation: `utils/cn`, `primitives/formControlContext`. External: `lucide-react` icons.
