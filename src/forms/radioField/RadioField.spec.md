# RadioField

## Purpose
Radio + right-side label + optional description, wrapped in a `<label>`. Mirror of `CheckboxField`.

## Props
| Name | Type | Default |
|---|---|---|
| `label` | `ReactNode` | — (required) |
| `description` | `ReactNode` | — |
| ...rest | `RadioProps` | — |

## Dependencies
Foundation: `utils/cn`, `hooks/useId`. Same-domain: `Radio`.
