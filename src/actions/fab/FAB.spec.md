# FAB

## Purpose
Floating Action Button — fixed-position circular shadowed button. Material/Mobile pattern for the most prominent action on a screen.

## Props
| Name | Type | Default | Required |
|---|---|---|---|
| `aria-label` | `string` | — | yes |
| `variant` | `'primary' \| 'secondary' \| 'destructive'` | `'primary'` | no |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | no |
| `position` | `'bottom-right' \| 'bottom-left' \| 'bottom-center' \| 'top-right' \| 'top-left'` | `'bottom-right'` | no |

## Dependencies
Foundation: `utils/cn`, `tailwind-variants`.
