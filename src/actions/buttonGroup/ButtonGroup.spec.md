# ButtonGroup

## Purpose
Group Button / IconButton children into a visually connected row or column.

## Props
| Name | Type | Default |
|---|---|---|
| `orientation` | `'horizontal' \| 'vertical'` | `'horizontal'` |
| `attached` | `boolean` | `true` |

## Dependencies
Foundation: `utils/cn`. Children: any clickable element (Button/IconButton/Link).
