# ButtonGroup

## Purpose
Group Button / Link / ToggleButton children into a visually connected row or column. For icon-only members, pass `<Button shape="square">` or `<Button shape="circle">`.

## Props
| Name | Type | Default |
|---|---|---|
| `orientation` | `'horizontal' \| 'vertical'` | `'horizontal'` |
| `attached` | `boolean` | `true` |

## Dependencies
Foundation: `utils/cn`. Children: any clickable element (Button/Link/ToggleButton).
