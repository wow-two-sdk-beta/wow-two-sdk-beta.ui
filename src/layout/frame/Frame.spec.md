# Frame

## Purpose
Bordered shell with padding + radius — `Card`'s visuals without the slot semantics. Use when you want a contained surface for free-form content.

## Props
| Name | Type | Default |
|---|---|---|
| `padding` | `'0' \| '2' \| '3' \| '4' \| '6' \| '8'` | `'4'` |
| `radius` | `'none' \| 'sm' \| 'md' \| 'lg'` | `'md'` |
| `surface` | `'card' \| 'muted' \| 'transparent'` | `'card'` |
| `bordered` | `boolean` | `true` |

## Dependencies
Foundation: `utils/cn`.
