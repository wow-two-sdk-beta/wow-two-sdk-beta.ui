# Spacer

## Purpose
Flexible empty box. Defaults to `flex: 1` for use inside `HStack`/`VStack`. With `size`, becomes a fixed-width or -height gap.

## Props
| Name | Type | Default |
|---|---|---|
| `size` | `number \| string` | — |
| `axis` | `'horizontal' \| 'vertical'` | `'horizontal'` |

## Dependencies
Foundation: `utils/cn`.
