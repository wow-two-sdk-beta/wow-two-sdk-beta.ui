# Inline

## Purpose
Wrapping horizontal row with consistent gap. For chips/tag rows, inline action groups.

## Props
| Name | Type | Default |
|---|---|---|
| `gap` | `'0' \| '1' \| '2' \| '3' \| '4' \| '6' \| '8'` | `'2'` |
| `align` | `'start' \| 'center' \| 'end' \| 'baseline'` | `'center'` |

## Dependencies
Foundation: `utils/cn`.
