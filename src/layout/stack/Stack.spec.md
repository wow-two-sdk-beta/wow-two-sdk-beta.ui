# Stack

## Purpose
Flex container with `direction`, `gap`, `align`, `justify`, `wrap` variants. Default direction `column`. Use as the universal layout primitive.

## Props
| Name | Type | Default |
|---|---|---|
| `direction` | `'row' \| 'column' \| 'row-reverse' \| 'column-reverse'` | `'column'` |
| `align` | `'start' \| 'center' \| 'end' \| 'stretch' \| 'baseline'` | — |
| `justify` | `'start' \| 'center' \| 'end' \| 'between' \| 'around' \| 'evenly'` | — |
| `gap` | `'0' \| '1' \| '2' \| '3' \| '4' \| '5' \| '6' \| '8' \| '10' \| '12'` | `'4'` |
| `wrap` | `'wrap' \| 'nowrap' \| 'wrap-reverse'` | — |
| `as` | `ElementType` | `'div'` |

## Dependencies
Foundation: `utils/cn`, `tailwind-variants`.
