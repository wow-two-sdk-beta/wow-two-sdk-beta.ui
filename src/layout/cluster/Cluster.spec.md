# Cluster

## Purpose
Centered wrapping row — auth pages, hero CTA pairs, footer link groups. Like `Inline` but `justify-center` by default.

## Props
| Name | Type | Default |
|---|---|---|
| `gap` | `'2' \| '3' \| '4' \| '6' \| '8'` | `'4'` |
| `justify` | `'start' \| 'center' \| 'end'` | `'center'` |

## Dependencies
Foundation: `utils/cn`.
