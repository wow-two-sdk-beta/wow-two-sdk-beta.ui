# DescriptionList

## Purpose
Semantic `<dl>` for label-value pairs — property panels, settings, info cards. `inline` for tabular layout, `stacked` for vertical pairs.

## Props
| Name | Type | Default |
|---|---|---|
| `items` | `{ label, value }[]` | — (required) |
| `layout` | `'inline' \| 'stacked'` | `'inline'` |
| `density` | `'sm' \| 'md' \| 'lg'` | `'md'` |

## Dependencies
Foundation: `utils/cn`.
