# AvatarGroup

## Purpose
Stacked row of `Avatar` children with overlap and a "+N" overflow tile.

## Props
| Name | Type | Default |
|---|---|---|
| `max` | `number` | unlimited |
| `size` | Avatar size | `'md'` |
| `overlap` | Tailwind class | `'-ml-2'` |

## Dependencies
Foundation: `utils/cn`. Same-domain: `Avatar`.
