# Badge

## Purpose
Static pill — status / category / count. For dismissible / interactive variants use `Tag`.

## Props
| Name | Type | Default |
|---|---|---|
| `variant` | `'neutral' \| 'brand' \| 'success' \| 'warning' \| 'danger' \| 'info' \| 'outline'` | `'neutral'` |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` |

## Dependencies
Foundation: `utils/cn`, `tailwind-variants`.
