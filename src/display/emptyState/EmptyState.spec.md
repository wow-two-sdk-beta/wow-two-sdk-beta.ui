# EmptyState

## Purpose
Empty-list / no-results affordance. Centered icon + title + description + action slots.

## Props
| Name | Type | Default |
|---|---|---|
| `icon` | `ReactNode` | — |
| `title` | `ReactNode` | — (required) |
| `description` | `ReactNode` | — |
| `actions` | `ReactNode` | — |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` |

## Dependencies
Foundation: `utils/cn`. Same-domain: `Heading`, `Text`.
