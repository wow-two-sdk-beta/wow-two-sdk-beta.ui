# Status

## Purpose
Colored dot + text label. Use for server status, online presence, build state. Pair with `hasPulse` for "live" indication.

## Props
| Name | Type | Default |
|---|---|---|
| `tone` | `'success' \| 'warning' \| 'destructive' \| 'info' \| 'neutral'` | `'success'` |
| `hasPulse` | `boolean` | `false` |
| `children` | `ReactNode` | — |

## Dependencies
Foundation: `utils/cn`.
