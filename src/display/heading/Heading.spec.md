# Heading

## Purpose
Semantic `h1`–`h6` with independent visual size. Decoupling level (semantic) from size (visual) lets layouts use the right outline without forcing a visual cascade.

## Props
| Name | Type | Default |
|---|---|---|
| `level` | `1 \| 2 \| 3 \| 4 \| 5 \| 6` | `2` |
| `size` | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl' \| '2xl' \| '3xl' \| '4xl'` | `'lg'` |
| `weight` | `'normal' \| 'medium' \| 'semibold' \| 'bold'` | `'semibold'` |
| `align` | `'left' \| 'center' \| 'right'` | — |

## Dependencies
Foundation: `utils/cn`, `tailwind-variants`.
