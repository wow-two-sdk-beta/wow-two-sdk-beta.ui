# KeyboardShortcut

## Purpose
Render a sequence of `Kbd` keys with connectors. Saves the boilerplate of `<Kbd>⌘</Kbd> + <Kbd>K</Kbd>`.

## Props
| Name | Type | Default |
|---|---|---|
| `keys` | `string[]` | — (required) |
| `separator` | `string` | `'+'` |

## Dependencies
Foundation: `utils/cn`. Same-domain: `Kbd`.
