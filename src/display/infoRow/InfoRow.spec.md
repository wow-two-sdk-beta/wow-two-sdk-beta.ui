# InfoRow

## Purpose
Single label + value row with optional leading icon. Use inside cards / detail panels for one or two values; use `DescriptionList` for many.

## Props
| Name | Type | Default |
|---|---|---|
| `label` | `ReactNode` | — (required) |
| `value` | `ReactNode` | — (required) |
| `icon` | `ReactNode` | — |
| `layout` | `'inline' \| 'stacked'` | `'inline'` |

## Dependencies
Foundation: `utils/cn`.
