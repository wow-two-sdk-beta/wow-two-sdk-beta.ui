# NotificationDot

## Purpose
Tiny colored dot for unread/notification indicators. Optionally positioned absolutely over a parent.

## Props
| Name | Type | Default |
|---|---|---|
| `tone` | `'destructive' \| 'success' \| 'warning' \| 'info' \| 'primary' \| 'neutral'` | `'destructive'` |
| `size` | `'xs' \| 'sm' \| 'md'` | `'sm'` |
| `pulse` | `boolean` | `false` |
| `position` | `'top-right' \| 'top-left' \| 'bottom-right' \| 'bottom-left'` | none (inline) |

## Dependencies
Foundation: `utils/cn`.
