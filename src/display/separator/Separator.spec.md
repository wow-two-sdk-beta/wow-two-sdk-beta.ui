# Separator

## Purpose
Hairline divider. Horizontal or vertical. `decorative` controls a11y semantics — purely visual by default.

## Props
| Name | Type | Default |
|---|---|---|
| `orientation` | `'horizontal' \| 'vertical'` | `'horizontal'` |
| `decorative` | `boolean` | `true` |

## Dependencies
Foundation: `utils/cn`.
