# ToastSimple

## Purpose
Atomic toast card — visual only, accepts free-form `children`. Pair: ToastSimple (atom) + Toast (slotted molecule, L4) + Toaster (queue, L5).

## Props
| Name | Type | Default |
|---|---|---|
| `severity` | `'info' \| 'success' \| 'warning' \| 'danger' \| 'neutral'` | `'neutral'` |

## Dependencies
Foundation: `utils/cn`, `tailwind-variants`.
