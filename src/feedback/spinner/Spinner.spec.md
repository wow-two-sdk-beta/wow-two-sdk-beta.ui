# Spinner

## Purpose
Indeterminate loading spinner. Visually hidden text label provides AT context.

## Props
| Name | Type | Default |
|---|---|---|
| `size` | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl'` | `'md'` |
| `tone` | `'default' \| 'brand' \| 'muted' \| 'current'` | `'default'` |
| `label` | `string` | `'Loading'` |

## Dependencies
Foundation: `utils/cn`, `tailwind-variants`.
