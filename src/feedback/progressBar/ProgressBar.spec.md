# ProgressBar

## Purpose
Linear progress. Determinate when `value` is a number; indeterminate when omitted.

## Props
| Name | Type | Default |
|---|---|---|
| `value` | `number \| undefined` | — |
| `max` | `number` | `100` |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` |
| `tone` | `'brand' \| 'success' \| 'warning' \| 'danger' \| 'neutral'` | `'brand'` |
| `label` | `string` | — |

## Dependencies
Foundation: `utils/cn`, `tailwind-variants`.
