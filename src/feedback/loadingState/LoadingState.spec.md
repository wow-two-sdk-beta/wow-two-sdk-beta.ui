# LoadingState

## Purpose
Centered Spinner + title + description for full-section/page loads. Use `InlineSpinner` for narrower in-flow cases.

## Props
| Name | Type | Default |
|---|---|---|
| `title` | `ReactNode` | `'Loading…'` |
| `description` | `ReactNode` | — |
| `size` | `'sm' \| 'md' \| 'lg' \| 'xl'` | `'lg'` |

## Dependencies
Foundation: `utils/cn`. Same-domain: `Spinner`.
