# AlertSimple

## Purpose
Atomic styled alert container — colored box, free-form `children`. Use when you want a tinted callout without the structural rigor of `Alert` (L4). Pair: AlertSimple (atom) + Alert (slotted molecule).

## Props
| Name | Type | Default |
|---|---|---|
| `severity` | `'info' \| 'success' \| 'warning' \| 'danger' \| 'neutral'` | `'info'` |

## Dependencies
Foundation: `utils/cn`, `tailwind-variants`.
