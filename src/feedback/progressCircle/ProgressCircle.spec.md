# ProgressCircle

## Purpose
Circular progress indicator. SVG ring; rotates when indeterminate.

## Props
| Name | Type | Default |
|---|---|---|
| `value` | `number \| undefined` | — |
| `max` | `number` | `100` |
| `size` | `number` (px) | `40` |
| `thickness` | `number` (px) | `4` |
| `tone` | `'brand' \| 'success' \| 'warning' \| 'danger' \| 'neutral'` | `'brand'` |
| `label` | `string` | — |

## Dependencies
Foundation: `utils/cn`.
