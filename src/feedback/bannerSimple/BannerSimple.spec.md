# BannerSimple

## Purpose
Full-width tinted banner — top-of-app status broadcast. Free-form `children`. Pair: BannerSimple (atom) + Banner (slotted molecule).

## Props
| Name | Type | Default |
|---|---|---|
| `severity` | `'info' \| 'success' \| 'warning' \| 'danger' \| 'neutral'` | `'info'` |

## Dependencies
Foundation: `utils/cn`, `tailwind-variants`.
