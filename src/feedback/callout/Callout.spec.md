# Callout

## Purpose
Inline doc-style note — colored left rule, no fill. Quieter than `Alert`. Common for MDX `> Note:` patterns.

## Props
| Name | Type | Default |
|---|---|---|
| `severity` | `'info' \| 'success' \| 'warning' \| 'danger' \| 'neutral'` | `'info'` |
| `icon` | `ReactNode` | — |
| `title` | `ReactNode` | — |
| `children` | `ReactNode` | — |

## Dependencies
Foundation: `utils/cn`.
