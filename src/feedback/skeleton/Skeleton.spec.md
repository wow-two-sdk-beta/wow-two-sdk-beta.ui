# Skeleton

## Purpose
Pulsing placeholder block during loading. `shape` covers the three common cases (rect / text-line / circle); pass explicit `w-*`/`h-*` to size.

## Props
| Name | Type | Default |
|---|---|---|
| `shape` | `'rect' \| 'text' \| 'circle'` | `'rect'` |

## Dependencies
Foundation: `utils/cn`, `tailwind-variants`.
