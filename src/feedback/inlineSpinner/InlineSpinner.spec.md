# InlineSpinner

## Purpose
Spinner + label inline — for inside buttons, list rows, anywhere a brief loading affordance is needed.

## Props
| Name | Type | Default |
|---|---|---|
| `children` | `ReactNode` | `'Loading…'` |
| `size` | Spinner size | `'sm'` |
| `tone` | Spinner tone | `'default'` |

## Dependencies
Foundation: `utils/cn`. Same-domain: `Spinner`.
